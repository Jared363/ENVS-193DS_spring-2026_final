# -ENVS-193DS_spring-2026_final
Jared's Final for ES193: Stats for Environmental Studies. Spring 2026

## General information 

This is the repository for my final for ENVS 193: Stats for Environmental Studies. I took this course in the Spring 2026 quarter. This final includes three main problems. Problem 1 deals with the rephrasing of scientific communiation. Problem 2 deals with generalized linear model analysis of how Acacia tree height and species of a symbiotic ant influence the probability that a bird nest is found in the Acacia tree. Problem 3 contains the analysis of how my computer screen time might differ between weekdays and weekends. 


## Data and file overview

The code folder has the qmd code file and rendered qmd for this final. 

The data folder contains the data used which is comoprised of the bird nest data, and my computer screen time data for my personal data project.
